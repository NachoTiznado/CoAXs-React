# Getting started

    npm install
    npm start


Then open your [http://localhost:3000/](http://localhost:3000/) to see the app.



# Start to develop with hot-load

    npm run startdev

Then open your [http://localhost:3000/](http://localhost:3000/) to see the app.



# Compress and build the production bundle

    npm run build-prod

Then open your [http://localhost:3000/](http://localhost:3000/) to see the app.
